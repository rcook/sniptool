# sniptool by Richard Cook

Code snippet management tool

## Clone repository

```
git clone https://github.com/rcook/sniptool.git
```

## Run main script

```
script/sniptool
```

## Build package

```
script/setup build
```

## Test package

```
script/setup test
```

## Upload package

```
script/setup sdist upload
```
